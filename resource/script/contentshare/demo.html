ContentShare DemoA jQuery Plugin which lets you Share what you Like

Intro

ContentShare is a jQuery plugin which lets you share the exact content on a page which you actually want to share. It enables you to share sections of text on your social network rather the default text specified in the <meta> tags. By default, it supports Facebook and Twitter but you can add more networks easily.
